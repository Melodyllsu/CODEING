## 3. 构造函数  
- 函数若在class body内定义完成，便自动成为inline候选人，最后结果还是由编译器决定  
- 访问级别  
private：一般变量是类自己使用的   
public：一般函数是被外界调用的   
- 构造函数  
```c
//构造函数的独特初始化方法 
complex (double r = 0, double i = 0)//默认参数  
  :re (r), im(i)  //初始化，规范， 效率高  
{ }               //赋值
```
- 构造函数的重载  
如果构造函数有默认值，要注意重载时注意不能冲突。  
如以下两个函数就是冲突的：  
```c
complex (double r = 0, double i = 0)
  :re (r), im(i)  
  { }
complex ():re (0), im (0)
  { }

```
